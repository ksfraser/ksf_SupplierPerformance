# Supplier Performance Module for FrontAccounting

Supplier evaluation, performance tracking, and relationship management.

## Features

- **Supplier Evaluation**: Multi-criteria supplier evaluation system
- **Performance Metrics**: On-time delivery, quality, cost performance tracking
- **Scorecards**: Supplier performance scorecards and dashboards
- **Improvement Plans**: Supplier improvement action plans and tracking
- **Contract Management**: Supplier contract terms and compliance tracking
- **Risk Assessment**: Supplier risk assessment and mitigation
- **Analytics**: Supplier performance analytics and benchmarking

## Requirements

- FrontAccounting 2.4+
- PHP 8.0+

## Future Development
